## Dependencies:

MacOS:
- Homebrew

Linux:
- package manager for your distro (apt, dnf, pacman, yay, ...)

Common:
- git
- gcc
- g++
- make
- cmake
- nvim
- tmux

- ghostty
- zsh
- go
- tree-sitter-cli

- python3

## Installation
```bash
./bootstrap.sh

./symlink.sh
```

- `zsh/alias.zsh`: aliases
- `zsh/path.zsh:`: $PATH
- `zsh/config.zsh`: other configurations
